# osTicket Helpdesk Lab

## Overview
This project documents my setup of a fully functional IT helpdesk ticketing system using osTicket on an Ubuntu Server virtual machine. The goal was to simulate a real helpdesk environment and practice ticket management workflows used in enterprise IT support.

## Environment
- Ubuntu Server 24.04 LTS (Oracle VirtualBox)
- osTicket v1.18.1
- Apache2, MySQL, PHP 8.5
- Host OS: Windows 11

## What I Built
- Deployed a LAMP stack from scratch on Ubuntu Server
- Installed and configured osTicket with custom departments, SLA plans, and help topics
- Created agent accounts and end user accounts to simulate a real org structure
- Worked tickets through a full lifecycle: intake, triage, assignment, internal notes, user reply, and resolution

## Departments
- IT Support (Level 1)
- Systems Administration (Level 2)

## SLA Plans
- Sev-A: 1 hour response, 24/7
- Sev-B: 4 hour response, 24/7
- Sev-C: 8 hour response, business hours

## Help Topics
- Password Reset
- Hardware Issue
- Software Install Request
- Network Connectivity
- New Employee Onboarding

## Ticket Lifecycle Example
Submitted a software install request as an end user, classified and assigned it to a Level 1 agent, updated the SLA, added an internal tech note, replied to the user, and resolved the ticket.

## Skills Demonstrated
- Linux server administration
- Apache virtual host configuration
- MySQL database setup and user management
- IT service management (ITIL-aligned SLA tiers and ticket workflows)
- osTicket administration
